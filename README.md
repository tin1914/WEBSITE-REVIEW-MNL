# Review Center Finder System

## Project Overview
The Review Center Finder System is a web-based application designed to help students search, locate, and review review centers in their area. It provides a centralized platform for both students and review centers to manage and access relevant information efficiently.

## Project Structure
- `frontend/` - Contains all user interface and client-side code.
- `backend/` - Contains server-side logic, APIs, and routes.
- `database/` - Contains database scripts, schemas, and authentication logic.
